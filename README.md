## 📂 Installation Steps
### For Windows Users:
1. Double-click the downloaded `.exe` file.
2. Follow the setup instructions on the screen.
3. Once the installation process finishes, you can find nvim-strudel in your Start Menu.

### For macOS Users:
1. Open the `.dmg` file by double-clicking it.
2. Drag the nvim-strudel icon to your Applications folder.
3. Eject the `.dmg` file and launch nvim-strudel from your Applications.

### For Linux Users:

2. Open a terminal and navigate to the directory where you extracted the files.

## 🎤 How to Use nvim-strudel
1. **Open Neovim**: Launch Neovim from your system or terminal.
2. **Load nvim-strudel**: Once inside Neovim, type `:Strudel` to load the plugin.
3. **Create Music**: Use the simple commands provided in the interface to start crafting your live music. You can add instruments, effects, and experiment with different sounds.
